# Python Fundamentals

> A bridging course for the Applied Data Science certification

Six units covering the Level 2 Python proficiency required to enter the Applied Data Science certification programme. Delivered via Jupyter notebooks in GitHub Codespaces.

## New here? Start with course-overview

**[course-overview](https://github.com/python-fundamentals-course/course-overview)** is the
course landing page — it has the full unit index and a
[Getting Started guide](https://github.com/python-fundamentals-course/course-overview/blob/main/GETTING_STARTED.md)
for learners new to GitHub, Codespaces, or Jupyter notebooks. No installation
required — everything runs in the browser.
